# Bank Icons Update - מעבר לסמלי בנקים

## שינויים שבוצעו

### 1. החלפת מטבעות זהב/שחורות בסמלי בנקים 🏦

**לפני**:
- 🪙 מטבעות זהב (Gold) - מוסיפות נקודות (+10)
- ⚫ מטבעות שחורות (Black) - מורידות נקודות (-15)

**אחרי**:
- 🏛️ **בנק הפועלים** - מוסיף נקודות (+10) ✅
- 🏦 **בנק לאומי** - מוריד נקודות (-15) ❌
- 💰 **בנק מזרחי טפחות** - מוריד נקודות (-15) ❌
- 💳 **בנק דיסקונט** - מוריד נקודות (-15) ❌

---

## מבנה המשחק החדש

### כללים:
1. **4 מטבעות בנק הפועלים** - תמיד על המסך (מטבעות טובות)
2. **3 מטבעות בנקים אחרים** - מופיעות באקראי ונעלמות אחרי 2 שניות (מטבעות קנס)

### מכניקת המשחק:
- לחיצה על **בנק הפועלים** → +10 נקודות + רטט + מטבע חדש אחרי שניה
- לחיצה על **בנקים אחרים** → -15 נקודות + רטט + המטבע נעלם
- מטבעות הקנס מופיעות באקראי בין 3 הבנקים (לאומי, מזרחי, דיסקונט)
- מטבעות הקנס נעלמות אוטומטית אחרי 2 שניות עם אנימציה

## קבצים ששונו

### 1. **Coin.kt** - מודל המטבעות
```kotlin
enum class CoinType {
    BANK_HAPOALIM,   // בנק הפועלים - Good coin (adds points)
    BANK_LEUMI,      // בנק לאומי - Penalty coin
    BANK_MIZRAHI,    // בנק מזרחי טפחות - Penalty coin
    BANK_DISCOUNT    // בנק דיסקונט - Penalty coin
}
```

**פונקציות חדשות**:
- `getIcon(type: CoinType)` - מחזיר emoji לכל בנק
- `isPenaltyCoin(type: CoinType)` - בודק אם מטבע הוא קנס

### 2. **GameEngine.kt** - ליבת המשחק
**קבועים חדשים**:
```kotlin
MIN_HAPOALIM_COIN_COUNT = 4      // תמיד 4 מטבעות הפועלים
MIN_PENALTY_COIN_COUNT = 3       // תמיד 3 מטבעות קנס
HAPOALIM_RESPAWN_DELAY_MS = 1000L // הפועלים מופיע אחרי שניה
```

**לוגיקה משופרת**:
- `startGame()` - יוצר 4 הפועלים + 3 קנסות אקראיים
- `collectCoin()` - הפועלים מופיע מחדש אחרי שניה
- `startCoinMaintenance()` - שומר על מינימום 4+3 מטבעות
- `startCoinCleanup()` - מסיר מטבעות קנס ישנות (2+ שניות)

### 3. **CoinOverlay.kt** - תצוגת המטבעות
```kotlin
Text(
    text = com.keren.virtualmoney.game.Coin.getIcon(coin.type),
    fontSize = (48 * coin.scale).sp
)
```

**אנימציות מותאמות**:
- מטבעות קנס פועמות מהר יותר (500ms במקום 1000ms)
- מטבעות קנס דועכות לפני היעלמות
